2015年10月20日

第1回 LODチャレンジデー in 名古屋 2015 開催のおしらせ


LODチャレンジ2015実行委員会は、オープンデータを使って未来を考えるイベント「第1回 LODチャレンジデー in 名古屋 2015」を開催します。


諸外国に比べてすごい速さで進む少子高齢化に立ち向かうためには、介護や医療、育児など、家庭に関する身近な社会課題を避けて通れません。そのような身近な社会課題を扱うアプリケーションやサービスを作ろうと考えたとき、オープンデータの仕組みが力を発揮すると期待されています。少子高齢化社会をオープンデータでデザインするアイディアソンとハッカソンのイベントを開催します。


【第1回 LODチャレンジデー in 名古屋 2015】

開催日時： 2015年10月24日（土）10:30-17:45（10:00より受付開始）

開催場所： 名古屋工業大学 4号館1階110 産学官交流スペース

参 加 費： 無料

主　　催： LODチャレンジ2015実行委員会

共　　催： Code for Nagoya，名工大 研究協力会／産学官連携センター／研究シーズオープンデータとシビックテック研究会

詳　　細： http://peatix.com/event/122203
