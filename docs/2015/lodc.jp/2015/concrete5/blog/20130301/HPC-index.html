技術コラム第12回：International Open Data Day in Japan 開催

2013/03/01 nakayama
2013年2月23日(土)，世界各地でオープンデータをテーマにしたイベント「International Open Data Day」が開催されました．世界各都市で市民が集まり，政府や自治体が保有する公共データを活用したアプリケーションの開発や，公共データを利用したアイデアの共有，埋もれたデータの発掘や新しいデータづくりなどを通して，市民自らが課題解決に取り組むイベントです．   日本においても初めて「International Open Data Day Japan」が開催され，青森、会津若松、鯖江、東京、千葉、横浜、名古屋、福岡の8都市が参加し，ハッカソンやアイデアソンが行われました．LODチャレンジ実行委員会は本イベントの開催に賛同し，横浜，千葉，名古屋の3都市で後援するとともに，実行委員も各都市のイベントに参加しました．参加レポートをご覧いただくと，各都市の地域性や特色が感じられると思います．共通して言えることは，地域市民がデータ活用を進めたり，データづくりに取り組くむことで，価値ある新しいサービスが立ち上がり始めたことです．今後もこうした取り組みを続けることによって，データの共有や，データどうしのつながりが促進され，地域社会の課題解決に活用されることでしょう．
