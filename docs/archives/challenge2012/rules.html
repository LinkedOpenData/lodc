LODチャレンジ実行委員会会則

名称

第1条 会の名称を、LODチャレンジ実行委員会（以下、実行委員会という）とする。

目的

第2条 実行委員会は、Linked Open DataチャレンジJapan 2012 の開催と、本コンテストの開催を通してLinked Open Dataの技術的、社会的な啓発と普及、関係するコミュニティの醸成を行うことを目的とする。

構成と役員

第3条　実行委員会は，Linked Open DataチャレンジJapanコンテスト2012の開催の趣旨に賛同する関係者で構成する。

２，実行委員会には、次の役職をおく。
(1)実行委員長
(2)事務局長、事務局
(3)監事

３，実行委員長は、委員の互選で選ぶ。実行委員長は、事務局長と監事を指名する。

４，事務局を、慶應義塾大学環境情報学部 萩野研究室内に置く。なお事務局としての活動は、実行委員会の各委員が分担して実施することがある。

役員と事務局の職務

第4条 実行委員長は、Linked Open Data チャレンジJapan 2012の全てと、実行委員会、事務局を統括する。

２，監事は、実行委員会の会計を監査する。

３，事務局長は事務局を統括し、事務局は、Linked Open Data チャレンジJapan 2012に係わる庶務、広報など事務を行う。

会議

第5条　実行委員会の会議は、実行委員長または実行委員長からの指示に基づいて事務局長が招集する。

２，実行委員会の開催は、Linked Open Data チャレンジJapan 2012が開催される年度の2012年4月1日から2013年3月31日において毎月一回行われる。

３，実行委員会用のメーリングリストやグループウェアを開設し、メーリングリストやグループウェアを利用した電子的な意見の交換なども、会議に準じるものとする。

４，実行委員長は、必要に応じて実行委員から個別に意見を聴取する。

任期

第6条　役員の任期は、Linked Open Data チャレンジJapan 2012が開催される年度の1年間、2012年4月1日から2013年3月31日とする。

報酬

第7条　実行委員は、無報酬とする。

実行委員の仕事

第8条　実行委員の主なる仕事は、次の通りとする。

(1) 実行委員会の会議に参加すること。

(2) 実行委員会用のメーリングリストやグループウェアに参加し電子的に発言をすること。

(3) Linked Open Data チャレンジJapan 2012開催目的を達成するために必要な諸機関の連携を強める活動

(4) Linked Open Data チャレンジJapan 2012への参加者・応募者を増やすための説明会などの広報や技術普及活動

(5) Linked Open Data チャレンジJapan 2012を成功させるために必要となる会場などの提供に係わる活動

(6) その他、上記に関連する創意工夫に満ちた活動

経費

第9条 Linked Open Data チャレンジJapan 2012開催に必要とする経費は、本コンテストに賛同するプラチナスポンサー、ゴールドスポンサーからのスポンサー費を充てることとする。

附則

この規約は2012年4月1日から実施する。
