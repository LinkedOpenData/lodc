応募するアプリケーションの情報
アプリケーションの名称 ご当地なび「防災情報」
アプリケーションのURL https://play.google.com/store/apps/details?id=net.flowertourism.itournavi
アプリケーションの概略説明 「ご当地なび」アプリに、オープンデータ化された「防災情報」を組み込んだ。 普段使っているアプリなので、 「いざ」というときに、直ぐに防災に役立たせることができる。
アプリケーションの詳細説明 「ご当地なび」アプリに、オープンデータ化された防災情報を組み込んだ。
1）「半径500m」ボタンから避難所情報を知ることができる。
現在位置から、周辺500m／1km／2kmのエリアの
　避難所情報
　小説・アニメ情報
　観光情報
などを、まとめて知ることができる。
目的情報をタップして詳細情報を把握できる。
詳細情報に写真がないときは、ストリートビューを使ってビジュアルに見せることで、目標を分かりやすくしている。
また、目的地へ地図を使ってナビゲーションを行っている。

2）防災情報メニューから避難所情報を知ることができる。
詳細情報は、上記１）と同じ。

3）操作性
普段使っているアプリの中に、避難所情報を埋め込んだこと、
観光情報と同じ操作性で、避難所情報を知ることができること、
など、「いざ」というときに、直ぐに防災に役立たせることができる。
