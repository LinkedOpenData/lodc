応募するビジュアライゼーション作品の情報
ビジュアライゼーション作品の名称 大阪市の警察署・交番と犯罪発生地点の重ね合わせ
ビジュアライゼーション作品の概略説明 大阪市がオープンデータとして公開する警察署・交番の位置情報を使って警察施設の無い地域を地図上に可視化し、さらに「大阪府警察犯罪発生マップ」の犯罪発生地点を重ね合わせました。
ビジュアライゼーション作品の詳細説明 大阪市が公開するオープンデータの中から、警察署・交番の位置情報を「LODチャレンジ応募作品のSPARQLエンドポイント（試行版）」から取得して「dangerzone-sparql」を使って警察署・交番の無い地域をビジュアライズ化しました。

さらに「大阪府警察犯罪発生マップ」から、2014年12月14日～2015年1月14日に発生した犯罪発生地点を重ね合わせました。

警察アイコンは、警察署・交番の位置、半透明の赤い円はその地域に警察署・交番が無いこと、赤、黄緑、青、黄色は、そこで起きた犯罪の種類を示します。
(赤色：ひったくり、黄緑：路上強盗、青色：子供被害情報、黄色：自動車盗難）

犯罪発生地点は人手でマッピングしていますので、正確な位置とズレがあります。
投稿したビジュアライゼーション作品 http://uedayou.net/osakacrimemap/
ビジュアライゼーション作品で利用したデータセット 大阪市が公開しているオープンデータ（http://www.city.osaka.lg.jp/toshikeikaku/page/0000250227.html）と 「大阪府警察犯罪発生マップ」の犯罪発生位置のデータ（http://www.map.police.pref.osaka.jp/Public/）を使用しました。 大阪市のオープンデータは、LODチャレンジ2013データセット部門応募作品「大阪市の施設情報・防災情報のLOD」でLOD化されたデータを使用しています。
