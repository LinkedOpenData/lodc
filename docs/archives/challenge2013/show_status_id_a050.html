・内容
　京都観光を車椅子で巡るときの利用可能なトイレ情報です。
　車椅子利用者のニーズにもとづいて、取材・撮影をおこない、データ作成をおこないました。

　目的地を事前に把握でき、スムーズに辿り着けます。
