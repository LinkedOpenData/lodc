RDF Site Summary (RSS) 1.0

公式の仕様書

RSS 1.0 仕様書は2000年12月6日にリリースされました。

その仕様書はRSSが容易に拡張できるようにRSSモジュールのデザインのためのガイドラインと一緒に発行されました。

モジュール

URI http://purl.org/rss/1.0/modules/を使用してRSSモジュールのクラスを表現することができます。

以下はRSS-DEVワーキンググループの公式のモジュールです。

  • Dublin Core
  • Syndication
  • Content

スキーマ

RSSを使用することによってRDFスキーマが利用できます。 本文書にはコピーも埋め込まれています。

Leigh DoddsはRSS 1.0のSchematron Schema を作成しました。
